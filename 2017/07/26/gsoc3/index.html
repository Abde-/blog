GSOC devlog (26/7/2017)

2017-07-26
gsoc
3d, pysoy, rendering

Hello again!

This week I’ve been working in trying to fix what was wrong in the previous devlog. As far as I could, I’ve fixed the point of view of the main pass but still no shadows. I have to take more time to understand how the attributes in GLSL work. I’m taking a lot of time in this step…

Screenshot of the main pass working, but no shadows (yet). My main goal is to get this finally working…

See ya!
